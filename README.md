# Moorthy X — Premium Portfolio Website

A luxury-inspired personal portfolio website crafted for **Moorthy Xavier**, showcasing graphic design expertise, brand identity work, packaging design, UI/UX projects, and creative storytelling through an immersive glossy user experience.

## ✨ Overview

This portfolio is designed to feel less like a traditional resume and more like a premium digital experience. Inspired by modern design leaders such as Apple, Linear, Stripe, and Framer, it combines glassmorphism, ambient lighting, smooth animations, and elegant typography to create a memorable personal brand presence.
